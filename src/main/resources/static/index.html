Vill du transformera koordinater?

Vad kul, då kan du ju använda vårt lilla api som ger dig möjlighet att transformera mellan koordinatsystemen WGS 84, SWEREF 99 TM och RT 90.

Så här ser vårt enkla REST-api ut: 

http://api.minkoordinat.se/api/v1/transformera/{koorindatsystem}/{northing}/{easting}

Tillåtna värden för {koordinatsystem} är:
  • wgs84
  • sweref99tm
  • rt90
Koordinaterna anges med {northing} och {easting} och ska vara decimaltal med punkt som decimalavskiljare. Exempel: 
http://api.minkoordinat.se/api/v1/transformera/wgs84/60.67488/17.14127
Detta ger svaret: 
{
"sweref99tm":{
   "northing":6728481.009694073,
   "easting":616980.0031438438
 },
"rt90":{
   "northing":6729377.7996849585,
   "easting":1573031.0827265498,
   "x":6729377.7996849585,
   "y":1573031.0827265498
 },
"wgs84":{
   "lat":60.67488,
   "long":17.14127
 }
}
Kör hårt och var snäll... :-)
